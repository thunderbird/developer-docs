# About Thunderbird

Thunderbird is a popular open source email client that has been around for more than 20 years. If you are looking to download it and try it out, please see our main page: https://www.thunderbird.net.

While Thunderbird is a company, the Thunderbird community of volunteers is the backbone of the project. If you want to join this vibrant community and see how you can make a difference, continue to the [Getting Started Contributing page](thunderbird-development/getting-started.md).
